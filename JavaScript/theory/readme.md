### 生成器中yield和yield*的区别：
yield 与 yield* 的区别在于：yield 只是返回右值，而 yield* 则将函数委托（delegate）到另一个生成器（ Generator）或可迭代的对象（如字符串、数组和类数组 arguments，以及 ES6 中的 Map、Set 等）。
